# /remove
Remove an item that is in the Randl database.



## Authentication

`write`



## Method

`POST`



## Arguments

**api_key** `required`

Your API key for Randl API.


**uuid** `required`
  
The `uuid` of the item that is supposed to be removed.



## Error Codes

- `200` Ok

- `401` Unauthorized

- `500` Internal Server Error
